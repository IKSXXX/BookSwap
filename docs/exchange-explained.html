📚 BookSwap — построчный разбор Exchange

О чём этот код

ExchangeService — сервисный слой: вся бизнес-логика обмена книгами (создание заявки, смена статусов, передача владения, отзывы, пересчёт рейтинга).

ExchangeController — слой представления: принимает HTTP-запросы, готовит ViewModel для страниц и делегирует все изменения данных сервису.

Что было почищено в этой ревизии

  • ExchangeService.CreateAsync — убраны лишние запросы: проверка владельца шла отдельным BookOwners.AnyAsync, хотя книга уже загружена с Include(BookOwners). Предлагаемая книга грузится одним запросом с Include вместо двух.
  • ExchangeController.Details — вместо ~7 отдельных запросов один запрос с Include.
  • ExchangeController.Create (GET) — два запроса объединены в один; проверка владельца берётся из загруженных BookOwners.
  • Свойство IsSender удалено из ExchangeListItemViewModel — оно нигде не читалось во вьюхах.
